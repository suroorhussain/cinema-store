Description

CinemaStore is a simple UI built using React and Bootstrap Grid System. All external dependencies are included within the project directory.

Dependencies Included
  1. Bootstrap (v3.3.5).
  2. React (0.14.2).
  3. Lodash (v.3.10.1).
  4. Browser - for supporting ES6 in the browser.
Install
  1. React Developer Tools plugin for the Chrome web browser.
Usage

To run the application do: $ python -m SimpleHTTPServer within the project directory.

Visit http://127.0.0.1:8000/index.html to access the application.
